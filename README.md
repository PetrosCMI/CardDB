# CardDB
HTML Database of membership cards that doesn't have advertising or other app issues.<br>
Only tested with Chrome.<br>
Scanned cards are stored on the device.<br>
There is no backend server.<br>
Features: 
 - Scan front and optionally back of card<br>
 - Rename a card<br>
 - Delete a card<br>
 - Backup all cards to a json file.  Can be manually moved somewhere safe.
 - Restore/merge cards from a json file.<br>
 - App can be loaded via url: https:/petroscmi.github.io/CardDB/
